# **Научиться учиться (how-to-learn)**

## Данная работа представляет собой выполненное мной (хоть и с небольшим нарушением мягкого дедлайна) техническое задание по результатам второго спринта первого модуля курса обучения веб-разработке на образовательной платформе Яндекс.Практикум.

## Я реализовал техническое задание с помощью таких языков программирования, как HTML5 и CSS.

## Файловая структура проекта выполнена в соответствии с правилами Nested БЭМ.

## Планируемые доработки: 1. замена шрифтов; 2. добавление рабочих ссылок; 3. добавление анимации для логотипов из раздела "resources".
